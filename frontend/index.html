Key Features

Built to be honest.

📵
Zero Friction Onboarding
No sign-up form. No login screen. Device ID and FCM token generated silently on first launch. Citizen is registered in under 2 seconds without doing anything.
UX
🔋
OS-Level · ~2% Battery
We hand zone boundaries to the Android/iOS hardware chip directly. The app can be killed — geofencing still works. ~2% battery impact vs 40% for active JS polling.
TECH
⏱️
Dwell Time Filter
User must physically stay inside a zone for a minimum time (5 min) before any notification fires. Someone driving past or walking through quickly is completely ignored.
SMART
🔁
Smart Deduplication
Redis stores a key per user+zone with 24-hour TTL. No spam, even if they re-enter the zone 10 times. Key expires automatically so next-day visits trigger fresh alerts.
REDIS
✅
Verified Data Only
Every notification links to a verified MongoDB record: scheme name, sanctioned budget, amount spent, completion %, responsible leader. Claims become auditable facts.
TRUST
📶
Offline Resilient
Zone data cached locally after first load. If internet is lost, geofencing still works using last-fetched zones. Backend ping retried automatically on reconnect.
ROBUST
